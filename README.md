# AWS Bedrock AI Chatbot using Terraform & Lambda

## Project Overview

This project demonstrates a serverless AI-powered chatbot application using AWS Bedrock, AWS Lambda, Python, and Terraform.

The infrastructure is provisioned using Terraform, while AWS Lambda invokes Amazon Bedrock foundation models to generate AI responses dynamically.

---

## Architecture

User Request → AWS Lambda → Amazon Bedrock → AI Response

## Deployment Steps

1. Configure AWS CLI
2. Initialize Terraform
3. Create IAM Role and Lambda using Terraform
4. Deploy Lambda function
5. Integrate Bedrock API using Python Boto3
6. Test Lambda using AWS Console
